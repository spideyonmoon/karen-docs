# Welcome to Karen

**Karen** is a Telegram bot that rips lossless ALAC, Atmos, and AAC audio from Apple Music and delivers it directly to your Telegram chat.

## What Karen Does

- ✅ **Extracts lossless ALAC & Atmos** from Apple Music streams
- ✅ **Handles DRM decryption** via Frida-hooked Android emulator
- ✅ **Embeds metadata & cover art** automatically
- ✅ **Intelligent delivery** — Bot API for <50MB, MTProto for up to 2GB, Gofile fallback for oversized packages
- ✅ **Multi-account pooling** — distribute load across N Apple Music accounts
- ✅ **User profiles** — save rip preferences for zero-prompt downloads

## Architecture at a Glance

Karen is a **distributed multi-service system**:

```
Telegram User
    ↓ /dl command
    ↓
Karen Bot (Go)
    ↓ gRPC
    ↓
Wrapper-Manager Instances (one per Apple Music account)
    ├─ Frida-hooked Android emulator
    ├─ DRM decryption oracle
    └─ gRPC endpoints: Login, GetM3U8, Decrypt
```

- **Bot** — Telegram command handler, download orchestrator, uploader
- **Wrapper-Manager** — Android emulator with hooked Apple Music app for DRM decryption
- **Pool** — FIFO load balancer across all wrapper-manager instances
